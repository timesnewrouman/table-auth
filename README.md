# TableAuth
###### Версия 1.3
#### Демо https://timesnewrouman.github.io/table-auth

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
### Launches server part
```
npm run node
```

### Launches both front and server parts
```
npm run main
```

### Deploy script
```
deploy.sh
```
